Type Description
bool

TRUE if the current version should be validated via ValidateCurrent(Action<Result>), FALSE otherwise.
Remarks

Current version should be validated if: LastKnownGitHubVersion is unknown, default, or invalid, LastKnownGitHubVersionSetDate is unknown or is older than LastKnownGitHubVersionFreshTimeHours.

ValidateCurrent(Action<Result>)

Validate the current version stored in a file with the release version published on GitHub. It performs the async HTTP request. Provide results as a callback and update the config .

Parameters
Type Name Description
Action<Result> onComplete

The callback (with Results object) which will be called once the HTTP request is sent and got results, no matter correctness. If this method returns TRUE, then callback should always be called at some point. That callback is scheduled by a non-main thread, but will be performed on the main Unity thread anyway.
Returns
Type Description
bool

TURE if request has been sent and there is something to wait for. FALSE if it failed and there is nothing to wait for (the onComplete will not be called).
