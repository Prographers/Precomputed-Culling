ColorToFloatDivisor

Because all shaders are using float but colors in band of 0-255 int we need to divide index of the object to be a float 0-1. Keep default 32 unless your total amount of objects is above 262144 renderers or lower the value for less complex scenes if you know you will have less renderers to decrease possibility of float rounding error.
