ContentRoot

Gets reference to the ContentRoot visual element, which serves as the root for all window content. This Visual Element should be the only child of window Canvas (instanced template), or child of another instanced template which is the first child of Canvas.

Anyway, as long as it has the correct name 'ContentRoot', it will be found and cached.
